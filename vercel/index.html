About Gonnect 4

About Gonnect 4

Gonnect 4 is a digital implementation of the classic Connect Four board game with a twist: it features a perfect AI opponent written in pure Go, compiled in WebAssembly for web browsers.

Game Overview

Connect Four is a strategy game where two players take turns dropping colored pieces into a 7-column, 6-row board. The first player to align four pieces horizontally, vertically, or diagonally wins the game.

Features

  • Perfect AI: Challenge an AI opponent that plays flawlessly
  • Multiple Difficulty Levels:
    • Easy: Plays randomly
    • Hard: The more a column has a higher score, the more likely it is to be chosen
    • Perfect: Always plays the best possible move
  • Score Analysis: View column evaluation scores and AI recommendations. The higher the score, the better the move for the current player
  • Move Replay: Share and replay game sequences via URL parameters
  • Responsive Design: Play on desktop, tablet, or mobile

How to Play

Basic Gameplay

The game starts with Player 1 (blue pieces). Click on any column to drop a piece at the top—it will fall to the lowest available position.

Controls

  • Drop a Piece: Click anywhere on the board or use the arrow buttons
  • Change Players: Switch who goes first using the "Player 1" buttons
  • Select Opponent: Choose between Manual, Easy AI, Hard AI, or Perfect AI
  • Reset Game: Start a new game with the same settings
  • View Scores: Toggle column analysis to see AI evaluation of moves

Winning

Get four pieces in a row (horizontal, vertical, or diagonal) to win! The game ends in a draw if the board fills without a winner.

How It Works

AI Engine

The Perfect AI uses the Negamax algorithm with Alpha-Beta pruning to evaluate board positions and determine the optimal move. This ensures it plays perfectly or nearly perfectly against any opponent.

Algorithm and optimization overview

  • Negamax: Recursively evaluates all possible future moves
  • Alpha-Beta Pruning: Eliminates unnecessary branches to improve performance
  • Transposition table: Caches board positions to avoid recalculation
  • Bitboards: Efficiently represents the game state using bitwise operations
  • Move sorting: Orders moves with a heuristic to improve pruning effectiveness

Difficulty Levels

  • Easy: Makes random legal moves
  • Hard: Evaluates positions at shallow depth (~8 moves ahead)
  • Perfect: Full negamax evaluation (nearly unbeatable)
